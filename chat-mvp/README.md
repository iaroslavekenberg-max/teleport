# Chat MVP

Простой веб-мессенджер MVP для команды:
- личные сообщения
- группы
- каналы
- realtime через WebSocket
- шифрование сообщений в хранилище (AES-256-GCM)

## Быстрый старт

1. Откройте папку `chat-mvp`
2. Установите зависимости:
   - `npm.cmd install`
3. Создайте `.env` на основе `.env.example` (главное: `APP_ENCRYPTION_KEY`)
4. Запустите:
   - `npm.cmd start`
5. Откройте в браузере:
   - `http://localhost:8787`

## Важно про шифрование

- Нужен постоянный ключ `APP_ENCRYPTION_KEY` (64 hex-символа).
- Если ключ не задан, сервер создаст временный ключ и после перезапуска старые сообщения нельзя будет расшифровать.
